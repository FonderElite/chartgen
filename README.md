=> Execute ```./chartgen -c -s``` Then Specify What Type Of Chart.
(Start Generating Charts)

# Requirements 

Chart-Gen requires the BeautifulSoup library and requests, installation below

    ~$  pip install -r requirements.txt

# Executing the code

    ~$ python3 chartgen.py
